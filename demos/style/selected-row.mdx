# Styling Selected Row

import MaterialTable from "@material-table/core";
import { InfoBannerAboutGlobalVarsInDemos } from "../../src/components";

## Props

| Field              | Type                 | Description                          |
| ------------------ | -------------------- | ------------------------------------ |
| `options.rowStyle` | style object or func | style to apply in all rows cells     |
| `onRowClick`       | func                 | to handle row click (event, rowData) |

## Usage

```jsx
import MaterialTable from "@material-table/core";

<MaterialTable
  // ...
  onRowClick={(evt, selectedRow) => setSelectedRow(selectedRow.tableData.id)}
  options={{
    rowStyle: (rowData) => ({
      backgroundColor:
        selectedRow === rowData.tableData.id ? "#6ABAC9" : "#FFF",
    }),
  }}
/>;
```
