# Styling Rows

import MaterialTable from "@material-table/core";
import { InfoBannerAboutGlobalVarsInDemos } from "../../src/components";

## Props

| Field              | Type                 | Description                      |
| ------------------ | -------------------- | -------------------------------- |
| `options.rowStyle` | style object or func | style to apply in all rows cells |

## Usage

```jsx
import MaterialTable from "@material-table/core";

<MaterialTable
  // ...
  options={{
    rowStyle: {
      backgroundColor: "#6ABAC9",
    },
  }}
/>;
```
