# Styling Header & Cells

import MaterialTable from "@material-table/core";
import { InfoBannerAboutGlobalVarsInDemos } from "../../src/components";

## Props

| Field                | Type                 | Description                        |
| -------------------- | -------------------- | ---------------------------------- |
| `column.cellStyle`   | style object or func | style to apply in specified cells  |
| `column.headerStyle` | style object         | style to apply in all header cells |

## Usage

```jsx
import MaterialTable from '@material-table/core';

<MaterialTable
    // ...
    columns={[
      {
          title: 'Name', field: 'name',
          cellStyle: {
            backgroundColor: '#039be5',
            color: '#FFF'
          },
          headerStyle: {
            backgroundColor: '#039be5',
          }
        }
      }
    ]}
    options={{
      headerStyle: {
        backgroundColor: '#01579b',
        color: '#FFF'
      }
    }}
/>
```
